Noto Sans Nag Mundari is a design for the Indic Nag Mundari script.

Noto Sans Nag Mundari has multiple weights, contains 66 glyphs, and supports 63 characters from 2 Unicode blocks: Nag Mundari, Basic Latin.

Supported writing systems

Nag Mundari

Nag Mundari is an Indic alphabet, written left-to-right. Read more on ScriptSource, Unicode, Wikipedia, r12a.
